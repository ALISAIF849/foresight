Getting Started
Frontend

Simply open frontend/index.html in your browser to see the landing page.

Backend
cd backend
pip install -r requirements.txt

Create a .env file inside the backend folder:

ELEVENLABS_API_KEY=your_api_key_here

Then run:

python main.py
